# Процесс развертывания

### Требования

- Node.js 18+
- npm 9+

### Установка зависимостей

```bash
npm install
```

### Запуск приложения

```bash
npm start
```

### Запуск тестов

```bash
npm test
```

### Сборка образа

```bash
docker build -t code-editor-ai .
```

### Запуск в контейнере

```bash
docker run -it code-editor-ai
```
### CI/CD пайплайн

Пайплайн автоматически запускается при:
- Push в ветку `main`
- Создании Pull Request

**Этапы:**
1. Установка зависимостей
3. Запуск тестов
4. Сборка приложения
5. Docker сборка

## Сборка релиза

```bash
npm run build
```

Собранные файлы появятся в папке `dist/`.

## GitHub Releases

### Создание релиза

1. Создать тег: `git tag v1.0.0`
2. Отправить тег: `git push origin v1.0.0`
3. GitHub Actions создаст артефакт
4. Опубликовать в GitHub Releases

## Развертывание на production

1. Собрать приложение: `npm run build`
2. Создать Docker образ
3. Опубликовать в реестре
4. Развернуть на целевой системе
