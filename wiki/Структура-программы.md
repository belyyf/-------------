# Структура программы

## Общая структура

```
code-editor/
├── main.js              # Главный процесс Electron
├── preload.js           # Preload скрипт для IPC
├── index.html           # Основной HTML файл
├── renderer.js          # Рендерер процесс (логика редактора)
├── styles.css           # Стили приложения
├── package.json         # Конфигурация проекта
├── .eslintrc.js         # Конфигурация ESLint
├── jest.config.js       # Конфигурация Jest
├── Dockerfile           # Docker конфигурация
├── .gitignore           # Игнорируемые файлы
├── .dockerignore        # Игнорируемые для Docker файлы
├── .github/
│   └── workflows/
│       └── ci-cd.yml    # GitHub Actions workflow
├── tests/
│   ├── functions.js     # Тестируемые функции
│   └── functions.test.js # Unit-тесты
└── wiki/
    └── *.md             # Wiki документация
```

## Модули

### main.js

Главный процесс Electron. Отвечает за:
- Создание окна приложения
- Настройку IPC обработчиков для ИИ-функций
- Управление жизненным циклом приложения

**Функции:**
- `createWindow()` — создание главного окна
- IPC обработчики:
  - `ai-improve-text` — улучшение текста
  - `ai-rewrite-text` — перефразирование
  - `ai-fix-errors` — исправление ошибок
  - `ai-translate-text` — перевод

### preload.js

Preload скрипт для безопасного взаимодействия между главным и рендерер процессами.

**API:**
- `window.electronAPI.improveText(text)`
- `window.electronAPI.rewriteText(text)`
- `window.electronAPI.fixErrors(text)`
- `window.electronAPI.translateText(text)`

### renderer.js

Логика интерфейса приложения.

**Компоненты:**
- Управление форматированием (жирный, курсив, подчеркнутый)
- Выбор шрифта и размера
- Подсчёт символов и слов
- Обработка ИИ-кнопок

### index.html

HTML-разметка приложения:
- Toolbar — панель инструментов
- Editor — область редактора
- AI Panel — панель ИИ-ассистента
- Statusbar — статусная строка

### styles.css

Стили приложения в тёмной теме.
