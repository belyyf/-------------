# Технические решения

### Electron Architecture

Приложение использует архитектуру Electron с разделением на:
1. **Главный процесс** — управление окном, системные операции
2. **Рендерер процесс** — пользовательский интерфейс
3. **Preload скрипт** — безопасная связь между процессами

### Electron
- **Причина выбора:** Кроссплатформенность, возможность использования веб-технологий
- **Версия:** 28.x

### JavaScript (ES6+)
- **Причина выбора:** Универсальность, большая экосистема
- **Особенности:** Async/await для асинхронных операций

### Jest
- **Причина выбора:** Популярный фреймворк для тестирования JavaScript
- **Конфигурация:** Node.js окружение

### Docker
- **Причина выбора:** Стандартизация окружения сборки
- **Базовый образ:** node:18-alpine

## Структура данных

### Состояние редактора

```javascript
{
  text: string,           // Текст в редакторе
  fontFamily: string,     // Выбранный шрифт
  fontSize: number,       // Размер шрифта
  isBold: boolean,        // Жирный
  isItalic: boolean,      // Курсив
  isUnderline: boolean,   // Подчеркнутый
  charCount: number,      // Количество символов
  wordCount: number       // Количество слов
}
```

## Обработка ошибок

```javascript
try {
  const result = await window.electronAPI.improveText(text);
  showResult(result);
} catch (error) {
  showResult(`❌ Ошибка: ${error.message}`);
}
```
