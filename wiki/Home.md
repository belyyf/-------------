## О проекте

Редактор кода с ИИ-ассистентом — это десктопное приложение, разработанное на Electron, которое предоставляет возможности редактирования текста с функциями форматирования и ИИ-помощником.

### Основные возможности

1. **Редактирование текста** — написание, вставка, копирование
2. **Форматирование** — жирный, курсив, подчеркнутый, выбор шрифта и размера
3. **ИИ-ассистент** — улучшение текста, перефразирование, исправление ошибок, перевод

### Архитектура

Приложение состоит из двух основных модулей:
- **Главный процесс (main.js)** — управление окном, IPC обработчики
- **Рендерер процесс (renderer.js)** — логика интерфейса и взаимодействие с пользователем

### Технологии

- Electron 28.x
- JavaScript (ES6+)
- Jest (тестирование)
- Docker (контейнеризация)
- GitHub Actions (CI/CD)
