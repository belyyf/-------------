# Инструкция пользователя

## Запуск приложения

1. Установите зависимости: `npm install`
2. Запустите приложение: `npm start`

## Интерфейс

```
┌─────────────────────────────────────────────────────┐
│  [Шрифт ▼] [Размер ▼]  [B] [I] [U]                 │  ← Toolbar
├───────────────────────────┬─────────────────────────┤
│                           │  ИИ-ассистент           │
│                           │                         │
│    Область редактора      │  [✨ Улучшить текст]    │
│                           │  [🔄 Другими словами]   │
│                           │  [✅ Исправить ошибки]  │
│                           │  [🌐 Перевести]         │
│                           │                         │
│                           │  [Результат]            │
├───────────────────────────┴─────────────────────────┤
│  Символов: 0  │  Слов: 0                            │  ← Statusbar
└─────────────────────────────────────────────────────┘
```

## Основные функции

### Редактирование текста

1. Кликните в область редактора
2. Начните вводить текст
3. Используйте Ctrl+C / Ctrl+V для копирования/вставки

### Форматирование

**Жирный шрифт:**
1. Выделите текст (планируется)
2. Нажмите кнопку **B** на панели инструментов

**Курсив:**
1. Нажмите кнопку *I*

**Подчеркнутый:**
1. Нажмите кнопку U

**Изменение шрифта:**
1. Выберите шрифт из выпадающего списка

**Изменение размера:**
1. Выберите размер из выпадающего списка

### ИИ-функции

**Улучшить текст:**
1. Введите текст в редактор
2. Нажмите кнопку "✨ Улучшить текст"
3. Результат появится в ИИ-панели

**Другими словами:**
1. Введите текст
2. Нажмите "🔄 Другими словами"

**Исправить ошибки:**
1. Введите текст
2. Нажмите "✅ Исправить ошибки"

**Перевести на английский:**
1. Введите текст
2. Нажмите "🌐 Перевести на английский"

## Статусная строка

- **Символов:** количество символов в тексте
- **Слов:** количество слов в тексте

## Горячие клавиши (планируется)

- Ctrl+B — жирный
- Ctrl+I — курсив
- Ctrl+U — подчеркнутый
- Ctrl+S — сохранить
