# Code Editor with AI Assistant

Редактор кода с ИИ-ассистентом и форматированием текста, разработанный на Electron.

## Функционал

### Основные возможности
- ✏️ Написание, вставка и копирование текста
- 📝 Форматирование текста (жирный, курсив, подчеркнутый)
- 🔤 Выбор шрифта и размера
- 🤖 ИИ-ассистент для работы с текстом

### ИИ-функции
- **Улучшить текст** — улучшение стиля и читаемости
- **Другими словами** — перефразирование текста
- **Исправить ошибки** — проверка и исправление ошибок
- **Перевести на английский** — перевод текста

## Установка

```bash
npm install
```

## Настройка ИИ

### Вариант 1: Hugging Face API (рекомендуется)

1. Получи API токен: https://huggingface.co/settings/tokens
2. Создай токен типа "Read"
3. Вставь в файл `.env`: `HF_API_KEY=твой_токен_здесь`

**Преимущества:**
- ✅ Полностью бесплатно
- ✅ Простая активация
- ✅ Не нужна верификация

### Вариант 2: Groq API

1. Получи API ключ: https://console.groq.com/keys
2. Вставь в файл `.env`: `GROQ_API_KEY=твой_ключ_здесь`

**Преимущества:**
- ✅ Очень быстро
- ✅ 30 запросов/минуту

### Вариант 3: Google Gemini API

1. Получи API ключ: https://aistudio.google.com/apikey
2. Вставь в файл `.env`: `GEMINI_API_KEY=твой_ключ_здесь`

**Преимущества:**
- ✅ Качественные ответы
- ✅ 60 запросов/минуту, 1500/день

## Структура проекта

```
├── main.js          # Главный процесс Electron
├── preload.js       # Preload скрипт для IPC
├── index.html       # Основной HTML файл
├── renderer.js      # Рендерер процесс (логика редактора)
├── styles.css       # Стили приложения
├── tests/           # Unit-тесты
├── .github/         # GitHub Actions workflows
└── Dockerfile       # Docker конфигурация
```
