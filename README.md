# Code Editor with AI Assistant

Редактор кода с ИИ-ассистентом и форматированием текста, разработанный на Electron.

## Функционал

### Основные возможности
- Написание, вставка и копирование текста
- Форматирование текста (жирный, курсив, подчеркнутый)
- Выбор шрифта и размера
- ИИ-ассистент для работы с текстом

### ИИ-функции
- **Улучшить текст** — улучшение стиля и читаемости
- **Другими словами** — перефразирование текста
- **Исправить ошибки** — проверка и исправление ошибок
- **Перевести на английский** — перевод текста

## Установка

```bash
npm install
```
## Запуск

```bash
npm start
```

## Сборка

```bash
npm run build
```

## Тестирование

```bash
npm test
```

## Структура проекта

```
├── main.js          # Главный процесс Electron
├── preload.js       # Preload скрипт для IPC
├── index.html       # Основной HTML файл
├── renderer.js      # Рендерер процесс (логика редактора)
├── styles.css       # Стили приложения
├── tests/           # Unit-тесты
├── .github/         # GitHub Actions workflows
└── Dockerfile       # Docker конфигурация
```

## Технологии

- Electron — фреймворк для десктопных приложений
- JavaScript — язык программирования
- Jest — фреймворк для тестирования
- Docker — контейнеризация
- GitHub Actions — CI/CD
