## ВВЕДЕНИЕ

### Цель проекта
Произвести разработку программного продукта «Редактор кода с ИИ-ассистентом» в соответствии со стандартной технологией разработки программного обеспечения и с использованием современных инструментов разработки.

### Описание продукта
Разработанное приложение представляет собой десктопный редактор текста с возможностями форматирования (жирный, курсив, подчеркнутый, выбор шрифта и размера) и интеграцией ИИ-функций для улучшения текста (улучшение, перефразирование, исправление ошибок, перевод).

### 4.3. Сценарии использования

#### Сценарий 1: Форматирование текста
1. Пользователь вводит текст
2. Выделяет текст (планируется)
3. Нажимает кнопку форматирования (B/I/U)
4. Текст применяется форматирование

#### Сценарий 2: ИИ-улучшение
1. Пользователь вводит текст
2. Нажимает кнопку ИИ-функции
3. Система обрабатывает текст
4. Результат отображается в панели

#### Сценарий 3: Изменение шрифта
1. Пользователь выбирает шрифт из списка
2. Текст в редакторе отображается выбранным шрифтом

### 6.2. Ключевые решения

#### IPC коммуникация
```javascript
// main.js
ipcMain.handle('ai-improve-text', async (event, text) => {
  return `Улучшенный текст: ${text}`;
});

// preload.js
contextBridge.exposeInMainWorld('electronAPI', {
  improveText: (text) => ipcRenderer.invoke('ai-improve-text', text),
});

// renderer.js
const result = await window.electronAPI.improveText(text);
```
