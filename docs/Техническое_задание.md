## на разработку программного продукта
## «Редактор кода с ИИ-ассистентом»

---

### 1. ОБЩИЕ СВЕДЕНИЯ

**Наименование проекта:** Редактор кода с ИИ-ассистентом (Code Editor with AI Assistant)

**Заказчик:** Учебный проект по дисциплинам «Технология разработки программного обеспечения» и «Инструментальные средства разработки программного обеспечения»

**Разработчик:** Студент (единый исполнитель)

**Дата составления:** 2026 г.

---

### 2. НАЗНАЧЕНИЕ И ЦЕЛИ РАЗРАБОТКИ

**Назначение:**
Программный продукт представляет собой десктопное приложение для редактирования текста с возможностями форматирования и интеграции ИИ-функций для улучшения качества текста.

**Цели разработки:**
- Создание удобного инструмента для редактирования текста
- Интеграция ИИ-помощника для улучшения текста
- Освоение технологий разработки десктопных приложений на Electron
- Применение современных практик разработки (CI/CD, тестирование, контейнеризация)

### 4. ТРЕБОВАНИЯ К ИНТЕРФЕЙСУ

#### 4.1. Общий вид интерфейса

```
┌────────────────────────────────────────────────────────────┐
│  [Шрифт ▼] [16 ▼]  │  [B] [I] [U]                         │
├────────────────────┬───────────────────────────────────────┤
│                    │  ИИ-ассистент                         │
│                    │                                       │
│   Текстовый        │  [✨ Улучшить текст]                 │
│   редактор         │  [🔄 Другими словами]                │
│                    │  [✅ Исправить ошибки]               │
│                    │  [🌐 Перевести на английский]        │
│                    │                                       │
│                    │  ┌─────────────────────────┐         │
│                    │  │ Результат обработки     │         │
│                    │  └─────────────────────────┘         │
├────────────────────┴───────────────────────────────────────┤
│  Символов: 0  │  Слов: 0                                   │
└────────────────────────────────────────────────────────────┘
```

#### 4.2. Элементы управления

**Верхняя панель (Toolbar):**
- Выпадающий список выбора шрифта (Arial, Times New Roman, Courier New, Georgia, Verdana)
- Выпадающий список выбора размера (12, 14, 16, 18, 20, 24)
- Кнопка «B» — жирный шрифт
- Кнопка «I» — курсив
- Кнопка «U» — подчеркнутый

**Правая панель (AI Panel):**
- Кнопка «Улучшить текст»
- Кнопка «Другими словами»
- Кнопка «Исправить ошибки»
- Кнопка «Перевести на английский»
- Область отображения результата

**Нижняя панель (Statusbar):**
- Счётчик символов
- Счётчик слов

Приложение должно иметь модульную архитектуру с разделением на:
