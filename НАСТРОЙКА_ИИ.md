# Настройка Hugging Face API

## Шаг 1. Получение API токена

1. Перейди на сайт: https://huggingface.co/settings/tokens
2. Войди или зарегистрируйся (можно через Google/GitHub)
3. Нажми **"New token"**
4. Введи название (например: "editor-ai")
5. Выбери тип: **Read** (чтение)
6. Нажми **"Generate token"**
7. Скопируй токен (начинается на `hf_...`)

## Шаг 2. Добавление токена в проект

1. Открой файл `.env` в корне проекта
2. Вставь токен после `HF_API_KEY=`

Пример:
```
HF_API_KEY=hf_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## Шаг 3. Проверка работы

1. Запусти приложение: `npm start`
2. Введи текст в редактор
3. Нажми любую ИИ-кнопку
4. Должен появиться ответ от Llama 3.1

## Модель

Используется **Meta Llama 3.1 8B Instruct** — качественная модель от Meta.

## Бесплатные лимиты

- **Бесплатно** с ограничениями
- Модель может загружаться при первом запросе (~30 сек)
- Лимиты зависят от нагрузки сервиса

## Преимущества Hugging Face

- 🆓 Полностью бесплатно
- 🔑 Простая активация (не нужна верификация)
- 🎯 Хорошее качество ответов
- 🌍 Много доступных моделей

## Если токен не работает

1. Проверь, что токен скопирован без пробелов
2. Убедись, что тип токена: **Read**
3. Перезапусти приложение
4. При первом запуске модель может загружаться 30-60 секунд
