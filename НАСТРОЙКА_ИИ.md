# Настройка Hugging Face API

## Шаг 1. Получение API токена

1. ПерейТи на сайт: https://huggingface.co/settings/tokens
2.  **"New token"**
3. тип: **Read** (чтение)
4. Нажмите **"Generate token"**
5. Скопируйте токен (начинается на `hf_...`)

## Шаг 2. Добавление токена в проект

1. Откройте файл `.env` в корне проекта
2. Вставьте токен после `HF_API_KEY=`

## Шаг 3. Проверка работы

1. `npm start`
2. Введите текст в редактор
3. Нажми любую ИИ-кнопку
4. Должен появиться ответ 

## Если токен не работает

1. Проверьте, что токен скопирован без пробелов
2. Убедитесь, что тип токена: **Read**
3. Перезапустите приложение
4. При первом запуске модель может загружаться 30-60 секунд
